Add or delete users on servers with Ubuntu or Debian.

Add and delete users on your server - Ubuntu, Debian

Difficulty: 1
Time: 10 mins

In this article you will learn to add and delete users on a server running either Debian or Ubuntu.

Add a new user

  1. Type in the following command:
    sudo adduser username
    Replace username with the one you want to create.
  2. Confirm your password.
  3. If needed, you can enter additional user details.
  4. Type "Y" to confirm the user information and create the new user record.

You have now created a new user.

If you are logged in as root, to delete a user record you can use the above command without sudo.

Grant sudo access to users

If you want to grant sudo access (administrative privileges) to your users, you can use the visudo command.

sudo visudo

This command allows you to edit the sudoers file. You can now add your user under the user privilege specification just below root.

Switch users

If you want to login as the new user you just created, you can use the su command.

su - username
Replace username is the user you want to delete.

This allows you to switch to the new user without logging out first. If you want to logout before logging in as the new user, you can use the exit command.

Delete an existing user

Type in the following command:

sudo deluser --remove-home username
Replace username with the user you want to delete.

You have now deleted the user and the user's home directory.

If you are logged in as root, to delete a user record you can use the above command without sudo.
