Setting up Ruby on Rails using RVM in Arch Linux, Ruby, Rails

Install Ruby on Rails using RVM - Arch Linux

Difficulty: 1

Time: 30 minutes

Ruby on Rails® is a well-known open source stack for web application development. With Ruby Version Manager (RVM) you can easily manage multiple Ruby application environments and switch between them in a most efficient way.

This article will assist you on how to install Ruby on Rails in Arch Linux using RVM, and assumes that you have sudo privileges to perform this task. You may want to visit the Initial Server Setup - Arch Linux article to setup your Arch Linux server.

Install RVM

  1. Update all the packages.
    sudo pacman -Sy
  2. Start with installing curl, if it's not already installed.
    sudo pacman -S curl
  3. Install RVM by typing:
    sudo curl -L get.rvm.io | bash -s stable
  4. After you have run the above command, there is one more dependency that needs to be fixed, so type the following command:
    sudo curl -sSL https://rvm.io/mpapis.asc | gpg2 --import -
    sudo curl -L https://get.rvm.io | bash -s stable
  5. Open .bashrc file by typing:
    sudo vim .bashrc
  6. Add the lines in this example at the end of the file.
    [[ -s "$HOME/.rvm/scripts/rvm" ]] && source "$HOME/.rvm/scripts/rvm"
  7. Close the session, log in back to apply the changes, and then install the dependencies of RVM.
    rvm requirements

Install Ruby

  1. Before you can install Ruby, you need to install Rails.
    sudo pacman -S libyaml
  2. Install Ruby with RVM.
    rvm install ruby
  3. The Rail asset pipeline needs JavaScript runtime to compile code. On Arch Linux, Node.js (a server-side JavaScript environment) is preferred. To install Node.js, type the following command:
    sudo pacman -S nodejs

Install RubyGems

  1. Install gems using RVM.
    rvm rubygems current

Setup Rails

This task will take some time, so have patience. Once completed, Ruby on Rails will be installed.

  1. To install Rails, type:
    gem install rails
  2. Check the installed versions of Ruby and Rails.
    ruby -v
    rails -v
  3. Create the project directory for your Rail project by typing the following commands:
    sudo mkdir -p /usr/local/src/rail_project
  4. Move to the newly created directory, and grant all the required access using following commands:
    cd /usr/local/src/rail_project
    sudo chmod 777 /usr/local/src/rail_project
    rails new HelloWorld
    cd HelloWorld
    rails server

    Your Rails server should be running now.

  5. You can bind your Rails application using the following command:
    rails server --binding=[your public IP]

    Your new application is ready and can be accessed by visiting your_public_IP_or_domain:3000 (3000 is default port).

Next steps
