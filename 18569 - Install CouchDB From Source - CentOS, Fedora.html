Setting up CouchDB from source on CentOS & Fedora, Erlang, and Spider Monkey.

Install CouchDB from Source - CentOS, Fedora

Difficulty: 4
Time: 30 minutes

CouchDB database is known to embrace the entire web and belongs to NoSQL family. This article will assist you to install CouchDB from source on CentOS and Fedora.

Install build tools and dependencies

  1. Update all packages.
    sudo yum -y update
  2. Install development-tool.
    sudo yum -y groupinstall "Development Tools"
    3. sudo yum -y install libicu-devel curl-devel ncurses-devel libtool libxslt fop java-1.6.0-openjdk java-1.6.0-openjdk-devel unixODBC unixODBC-devel openssl-devel
  3. Install EPEL repository using the following commands:
    sudo wget http://dl.fedoraproject.org/pub/epel/7/x86_64/e/epel- release-7-5.noarch.rpm
    sudo rpm -ivh epel-release-7-5.noarch.rpm

Install Erlang

  1. Firstly create a new directory.
    sudo mkdir Install_couchDB
  2. Go to that directory and compile Erlang from source.
    sudo wget http://www.erlang.org/download/otp_src_R14B01.tar.gz
    sudo tar -xvf otp_src_R14B01.tar.gz
    cd otp_src_R14B01
    sudo ./configure
    sudo make && make install

Install Mozilla SpiderMonkey

  1. Download Mozilla SpiderMonkey from the source.
    sudo wget http://ftp.mozilla.org/pub/mozilla.org/js/mozjs17.0.0.tar.gz
  2. Compile the Mozilla SpiderMonkey.
    sudo wget http://ftp.mozilla.org/pub/mozilla.org/js/mozjs17.0.0.tar.gz
    sudo tar -xvf mozjs17.0.0.tar.gz
    cd mozjs17.0.0/js/src/
    sudo ./configure
    sudo make && make install

Install CouchDB

  1. Download CouchDB package from source.
    sudo wget http://mirror.tcpdiag.net/apache/couchdb/source/1.6.1/apache-couchdb-1.6.1.tar.gz
    sudo tar -xvf apache-couchdb-1.6.1.tar.gz
    cd apache-couchdb-1.6.1
  2. Now, export the path of Erlang using the following commands:
    export ERL_TOP='pwd'
    export PATH=$ERL_TOP/bin:$PATH
  3. Now, compile the CouchDB.
    sudo ./configure --with-erlang=/usr/lib64/erlang/erts-5.10.4/include
    sudo make && make install

Setup CouchDB

  1. Create a CouchDB user and give some permissions using following command:
    sudo adduser --no-create-home couchdb
    sudo chown -R couchdb:couchdb /usr/local/var/lib/couchdb /usr/local/var/log/couchdb /usr/local/var/run/couchdb
  2. Create a link for the couchdb init script to /etc/init.d.
    sudo ln -sf /usr/local/etc/rc.d/couchdb /etc/init.d/couchdb
  3. To start CouchDB automatically at boot, add and enable the init script using chkconfig:
    sudo chkconfig --add couchdb
    sudo chkconfig couchdb on
  4. Open the configuration file of CouchDB.
    sudo vim /usr/local/etc/couchdb/local.ini
  5. To access CouchDB from the web, go to the Httpd section and make the following changes: 
    	[httpd]
	port = 5984
	bind_address = 0.0.0.0
  6. Now, restart the CouchDB services.
    sudo service couchdb restart
  7. To test CouchDB is running successfully, go to any web browser and type:

    http:Your public Ip:5984

  8. Once the server is up and running, you can access it in your browser at:

    http:Your public IP:5984/utils/
