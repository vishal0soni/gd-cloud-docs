Steps to install Ruby on Rails using Ruby Version Manager (RVM) in Fedora, Ruby, Rails

Install Ruby on Rails using RVM - Fedora

Difficulty: 1

Time: 30 minutes

Ruby on Rails™ is a well-known framework for web application development. The Ruby Version Manager (RVM) allows developers to easily manage multiple Ruby application environments and switch between them in a most efficient way.

This article explains on how to install Ruby on Rails in Fedora™ using RVM.

Prepare your system

Before you install Ruby on Rails, prepare your system with the required system software.
  1. Update the system software and access root by running the following command:
    sudo yum update
  2. You will use curl to install RVM. Install curl by typing:
    sudo yum install curl
  3. Install tar.
    sudo yum install tar

Update and install RVM and Ruby

  1. If your system doesn't have RVM already installed, type the following command to install RVM:
    sudo curl -L https://get.rvm.io | bash -s stable --ruby

    The backslash before curl prevents potential version conflicts and --ruby flag will install the latest version of Ruby.

  2. After you have run the above command, there are more dependencies that need to be fixed, so run the following commands again:
    sudo curl -sSL https://rvm.io/mpapis.asc | gpg2 --import -
    sudo curl -L https://get.rvm.io | bash -s stable
  3. To apply RVM, you need to execute the following command:
    source /home/UserName/.rvm/scripts/rvm
    If your system already has RVM installed, update it.
    source /home/UserName/.rvm/scripts/rvm

    If you've already installed these packages, you'll get a message.

Install Ruby

  1. Use RVM to install Ruby.
    rvm install ruby
  2. To check Ruby version, type:
    ruby -v
  3. Rail asset pipeline needs JavaScript runtime to compile code. On Ubuntu, Node.js (a server-side JavaScript environment) is preferred. To install Node.js, type the following command:
    sudo yum install nodejs

Install gems

Bundler is a tool to manage gems while developing and running Rails applications. Nokogiri is a dependency for many other gems.

  1. Update the gem manager.
    gem update -system
  2. Type following command to use Ruby with gemset global:
    rvm gemset use global
  3. Install both gems.
    gem install bundler
    gem install nokogiri

Install Rails

Finally, we are going to install Rails.

  1. Use gem to install the most recent stable release of Rails.
    gem install rails

    To check the version, type:

    rails -v
  2. Create the project directory for your Rail project, by typing the following commands:
    sudo mkdir -p /usr/local/src/rail_project
  3. Move to the newly created directory, and grant all the required access using following commands:
    cd /usr/local/src/rail_project
    sudo chmod 777 /usr/local/src/rail_project
    rails new HelloWorld
    cd HelloWorld
    rails server

    Now your Rails server should be running.

  4. You can bind your Rails application using following command:
    rails server --binding=[your public IP]

    Your new application is ready and can be accessed by visiting your_public_IP_or_domain:3000. 3000 is the default port.
