Setting up Jabber XMPP Server on CentOS and Fedora, XMPP

Install and configure Jabber XMPP server - CentOS, Fedora

Difficulty: 2

Time: 30 Minutes

This article explains the installation and configuration of Ejabber on CentOS and Fedora.

Jabber is a simple, mature and powerful messaging protocol which was developed to collaborate in real-time between two or more people, based on the text typed. Jabber is used by enterprises for internal communication, developers for improving programming skills, hosting provider for highly available and stable communication services, individual for chatting.

Note: You will need access to sudo privileges to complete all steps.

Download the Ejabber installer

  1. Run the following command to download Ejabber:

    wget http://www.process-one.net/downloads/downloads-action.php?file=/ejabberd/14.12/ejabberd-14.12-linux-x86_64-installer.run -o ejabberd-14.12-linux-x86_64-installer.run

  2. Change the file name

    mv downloads-action.php\?file\=%2Fejabberd%2F14.12%2Fejabberd-14.12-linux-x86_64-installer.run ejabberd-14.12-linux-x86_64-installer.run

    Change permissions

    sudo chmod +x ejabberd-14.12-linux-x86_64-installer.run

  3. Execute the Installer

    d. sudo ./ ejabberd-14.12-linux-x86_64-installer.run

Select language and accept the EULA

  1. After starting the installer, it will ask for language.
    select language English on Number [2]
  2. Accept the end user license agreement for the Ejabber installer by pressing Y

Select the installation directory and domain

  1. Press the Enter key to select the following installation directory
    Installation Directory [/opt/ejabberd-14.12]:

    If you want to keep the default directory, then press the enter key. Else, you can specify a different path as per your convenience.

  2. To selelct the domain, type FQDN or public IP or localhost or any internal IP from which we can access it
    ejabberd server domain [domain_name-centos]

Setup administrator credentials

  1. When prompted for the username, enter the username of your administrator.
    Administrator username [admin]:

  2. After choosing the username, next it will prompt for password. Enter a carefully chosen password.

Setup cluster

  1. After submitting the password, installer will prompt for cluster. Press 'N' as we don't require it now.
    promoted massage example Cluster [y/N]:N
  2. To finalize the installation, press Y. The installer will start the installation process and it will take some time to finish it.
    Do you want to continue?
  3. Next it will prompt to the view the readme file. Press Y or N
    View Readme file? [Y/n]: n

Start the Ejabber services

  1. Now our Ejabber protocol is ready to be used. Use the following command to start the Ejabber protocol:

    cd /opt/ejabberd-14.12/bin/ sudo ./ejabberdctl start
  2. Verify if the Ejabber service has been started by using the following command:
    sudo netstat -plunt | grep 5280

  3. If you get the output as shown below, then you have successfully installed the Jabber instant messaging service.

    tcp 0 0 0.0.0.0:5280 0.0.0.0:* LISTEN 27744/be

  4. Now open your browser and type http://your_ip:5280/ and hit Enter. It will prompt for authentication credentials, enter those credentials which you have entered during installation time. After successfully logging in, it will show the screen as shown below
