Installation of Joomla Console on your server.

Install Joomla Console to manage your Joomla site by command line - CentOS, Fedora

Difficulty: 1
Time: 10 minutes

Joomla Console is a command-line utility for the Joomla web CMS/application. It provides advanced functionality for power users who wish to manage their Joomla site via SSH. These advanced functions include rapidly creating new sites in Joomla, installing extensions, creating symlinks to custom code, version control with git, and many more - all through command line.

This article will cover how to properly install Joomla Console.

Download and install Joomla Console

  1. Install PHP, if you don't have it already.
    sudo yum update
    sudo yum install php php-mysql
  2. From your home directory, use the wget command to get the newest version of Joomla Console. Get this directly from the joomlatools.com GitHub, which ensures that it is always the most up-to-date version of Joomla Console.
    wget https://github.com/joomlatools/joomla-console/archive/master.zip
  3. Install unzip, and unzip the Joomla Console archive.
    sudo yum install unzip
    unzip -q master.zip
  4. For the ease of use, rename this directory "joomla-console."
    mv joomla-console-master/ joomla-console
    In your home directory, you should have a new directory named joomla-console.
  5. Remove the original master.zip file.
    rm master.zip
    Confirm with y.
  6. Update the permissions to the Joomla Console executable, so that can be executed by the server.
    chmod u+x ./joomla-console/bin/joomla

Install Composer

PHP Composer is a PHP dependency management suite that allows PHP to do the heavy lifting for you. It ensures that PHP knows which libraries to use when executing Joomla Console commands. This section will show how to get Composer, and then properly use Composer to install of Joomla Console.

  1. Use the curl command to grab the Composer files, and then pipe them to PHP to be executed.
    curl -sS https://getcomposer.org/installer | php
    You should see an output regarding the settings of Composer.
  2. Globally setup Composer.
    sudo mv composer.phar /usr/local/bin/composer
  3. Go to joomla-console directory and install Composer.
    cd joomla-console
    composer install
    You will see an output of different dependencies that are installed for Joomla Console. Composer will complete the installation of the Joomla Console dependencies.

Configure bash

  1. Switch back to your home directory.
    cd ~
  2. You need to add the proper alias to your .bashrc file in your home directory. If you've never edited your .bashrc file, or never installed any other command line utilities, you can add this to the .bashrc file with this one-line command:
    echo "alias joomla='~/joomla-console/bin/joomla'" >> .bashrc
  3. Make a similar edit to your .bash_profile file.
    echo "export PATH='$PATH:~/joomla-console/bin:/usr/local/bin'" >> .bash_profile
  4. Update the settings for bash so that it pulls the new settings from the new .bashrc and .bash_profile files.
    source .bashrc
    source .bash_profile
  5. Check the installation by running:
    joomla help

Conclusion

You have successfully installed the Joomla Console.
