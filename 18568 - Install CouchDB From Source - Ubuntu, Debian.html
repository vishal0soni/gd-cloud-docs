Install CouchDB From Source - Ubuntu, Debian

Difficulty: 4

Time: 30 minutes

CouchDB database is known to embrace the entire web and belongs to NoSQL family. This articles will assist you to install CouchDB from source in Ubuntu and Debian.

Step 1: Install dependencies

First of all we'll need to make our machine ready to install CouchDB.

  1. Update the packages for Ubuntu
    $ sudo apt-get udpate
  2. Setpup the Couch using following command
    $ sudo apt-get install -y build-essential
  3. To run erlang and other dependencies use following command
    $ sudo apt-get install -y erlang-base erlang-dev erlang-nox erlang-eunit
  4. Install other required libraries
    $ sudo apt-get install -y libmozjs185-dev libicu-dev libcurl4-gnutls-dev libtool

Step 2: Fetch the source

  1. Now we'll download the copy of source. Go to the directory where our code will reside
    $ cd /usr/local/src
  2. Fetch source using following command
    $ sudo curl -O http://apache.mirrors.tds.net/couchdb/source/1.6.1/apache-couchdb-1.6.1.tar.gz
  3. Extract the files
    $ sudo tar xvzf apache-couchdb-1.6.1.tar.gz
  4. Move to the new directory
    $ cd apache-couchdb-1.6.1.tar.gz

Step 3: Configure and install

To compile a newly spun CouchDB server, install the source code using following command

$ ./configure
$ sudo make && sudo make install

Step 4: Finalize the steps

  1. CouchDB runs as user process for couchdb, so create a new user with following command
    $ sudo adduser --disabled-login --disabled-password --no-create-home couchdb
  2. It will ask you for Real name, Room Number etc. you can skip these. Now give rights to the new user we created on the CouchDB's files
    $ sudo chown -R couchdb:couchdb /usr/local/var/log/couchdb /usr/local/var/lib/couchdb /usr/local/var/run/couchdb
  3. Install the CouchDB as service
    $ sudo ln -s /usr/local/etc/init.d/couchdb /etc/init.d
    $ sudo update-rc.d couchdb defaults
  4. Finally run CouchDB service
    $ sudo service couchdb start
