Installation of Joomla Console on your server.

Install Joomla Console to Manage Your Joomla Site by Command Line - Debian, Ubuntu

Difficulty: 1
Time: 10

Joomla Console is a command-line utility for the Joomla web CMS/application. It provides advanced functionality for power users who wish to manage their Joomla site via SSH. These advanced functions include creating new sites in Joomla rapidly, installing extensions, creating symlinks to custom code, version control with git, and many more - all through command line. This article covers how to install Joomla Console.

Download and install Joomla Console

  1. Install PHP unless you have it already:
    sudo apt-get update
    sudo apt-get install php5 php-pear php5-mysql
  2. From your home directory, use the wget command to get the newest version of Joomla Console. Get the version directly from the joomlatools.com GitHub to ensure that it is the most up-to-date.
    wget https://github.com/joomlatools/joomla-console/archive/master.zip
  3. Install unzip:
    sudo apt-get install unzip
  4. Unzip the Joomla Console archive:
    unzip -q master.zip
  5. For ease of use, rename this directory to "joomla-console":
    mv joomla-console-master/ joomla-console
    You now have a directory in your home directory named joomla-console.
  6. Remove the original master.zip file:
    rm master.zip
  7. Update the permissions to the Joomla Console executable so that can be executed by the server:
    chmod u+x ./joomla-console/bin/joomla

Install Composer

PHP Composer is a PHP dependency management suite that allows PHP to do the heavy lifting for you, ensuring that PHP knows which libraries to use when executing Joomla Console commands. This section shows how to get Composer and then to use Composer to complete the installation of Joomla Console.
  1. Use the curl command to grab the Composer files; then pipe them to PHP to be executed:
    curl -sS https://getcomposer.org/installer | php
    Output regarding Composer settings displays.
  2. Globally set up Composer:
    sudo mv composer.phar /usr/local/bin/composer
  3. Go to the joomla-console directory, and install Composer:
    cd joomla-console
    composer install
    Output of dependencies installed for Joomla Console displays. Composer completes the dependency installations for Joomla Console.

Configure bash

  1. Switch back to your home directory:
    cd ~
  2. Add the proper alias to your .bashrc file in your home directory.
    If you've never edited your .bashrc file or installed any other command line utilities, you can likely add the alias to the .bashrc file by using this one-line command:
    echo "alias joomla='~/joomla-console/bin/joomla'" >> .bashrc
  3. Make a similar edit to your .bash_profile file:
    echo "export PATH='$PATH:~/joomla-console/bin:/usr/local/bin'" >> .bash_profile
  4. Update bash settings so that it pulls the new settings from the new .bashrc and .bash_profile files:
    source .bashrc
    source .bash_profile
  5. Check the installation:
    joomla help

Conclusion

You have successfully installed the Joomla Console.
