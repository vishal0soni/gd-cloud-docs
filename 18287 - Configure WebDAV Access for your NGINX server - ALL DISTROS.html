Assumptions: you have a C compiler, Zlib Compression Library, PCRE, libexpat-dev, and OpenSSL installed

Preparation

  1. Create a directory for the NGINX source files
    sudo mkdir -p /source/nginx
  2. Navigate to source/nginx
    cd /source/nginx
  3. Get NGINX source from NGINX.org
    sudo wget http://nginx.org/download/nginx-1.8.0.tar.gz
  4. Extract NGINX Source
    sudo tar xfz nginx-1.8.0.tar.gz
  5. Get webDAV extension module from github
    sudo wget -c https://github.com/arut/nginx-dav-ext-module/archive/v0.0.2.zip -O nginx-dav-ext-module-v0.0.2.zip
  6. unzip it
    sudo unzip nginx-dav-ext-module-v0.0.2.zip
  7. Extract webDAV extension module
    tar xfz nginx-dav-ext-module-v0.0.2.zip

Compile NGINX

  1. Navigate to your newly-created nginx directory
    cd nginx-1.8.0
  2. Create a make.conf file
    sudo ./configure --user=nginx --group=nginx --prefix=/etc/nginx --sbin-path=/usr/sbin/nginx --conf-path=/etc/nginx/nginx.conf --error-log-path=/var/log/nginx/error.log --http-log-path=/var/log/nginx/access.log --pid-path=/var/run/nginx.pid --lock-path=/var/run/nginx.lock --with-http_ssl_module --with-pcre --with-http_dav_module --add-module=/source/nginx/nginx-dav-ext-module-0.0.2
  3. Create a makefile
    sudo make
  4. Install it
    sudo make install

Configure NGINX

NGINX supports dav_access,dav_methods, and dav_ext_methods in the http, server and location contexts. The folloing excerpts illustrate some of the possibilities:

  1. Global WebDAV access to all with all methods enabled 
    # http context
http {
...
dav_access group:rw all:r; 
dav_methods on;
dav_ext_methods PROPFIND OPTIONS;
...
server {
        listen 80 default_server;
...

location / {
             }
    }
server {
        listen 443 ssl;
...

location / {
             }
    }
}
  2. WebDAV access limited to a server block with selected methods (PUT DELETE MKCOL COPY) enabled (MOVE is excluded). 
    # server context
server {
        listen 80;
...
dav_access group:rw all:r;
dav_methods PUT DELETE MKCOL COPY;
dav_ext_methods PROPFIND OPTIONS;
...
location / {
             }
    }
  3. WebDAV access limited to a server block with selected methods enabled (PROPFIND OPTIONS are excluded by the lack of a dav_ext_methods directive). 
    # location context

location / {
...
dav_access group:rw all:r; 
dav_methods on;
...
             }
