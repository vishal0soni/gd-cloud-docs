Getting started with setting up (configuring) Git on all distros

Set Up Git - ALL DISTROS

Difficulty: 1
Time: 5 minutes

After installing Git, set up your GitHub user name and email address. Every Git commit uses these details. Without them, you get an error message (and have to revise) each comment you make.

Set up your GitHub user name and email address

Git comes with a configuration tool called git config that defines how it looks and works.

  1. Specify your user name and email:
    git config --global user.name "Dora the Explorer"
    git config --global user.email dora@explorer.com
  2. Check your settings:
    git config --list
    Git lists all the settings it can find. The output looks like this:
    user.name=Dora the Explorer
    user.email=dora@explorer.com
